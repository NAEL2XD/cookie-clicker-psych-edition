# How to download and update?

You'll need [git](https://git-scm.com/) for this!

Follow this step by step:

## Download

1. Go to one of the folders in your file explorer.
2. In the directory bar, type "cmd".
3. A command prompt should show up, now copy this: ```git clone https://github.com/NAEL2XD/cookie-clicker-psych-edition.git```.
4. Paste into your command prompt.
5. Wait.
6. If it's successful, drag it into your paych engine mods folder.
7. And voila, you're done!

## Update

1. Go to the `cookie-clicker-psych-edition` folder.
2. In the directory bar, type "cmd"
3. A command prompt should show up, now copy this: ```git pull origin main```
4. Paste into your command prompt.
5. Wait.
(if you get a fatal error, paste it again)
6. If you see a list that has green or red color, then voila you've updated it.
